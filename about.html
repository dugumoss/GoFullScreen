About GoFullScreen

This extension allows users to capture full-page screenshots and manage allowed domains. Users can add, remove, or clear the list of allowed domains through the settings page and request the necessary permissions upon confirmation.

This extension uses Html2Canvas and may not work on all web pages. If you have any suggestions or encounter any issues during use, please email me at here1271@163.com.

Thank you for using our extension!

If you find this extension helpful, I'm very glad. Independent development is not easy, and any support is greatly appreciated.
